Что нужно знать для вакансии HTML-верстальщика

HTML:

  • принцип строения HTML страницы
  • теги html, head, body и какие элементы в них могут находиться
  • основные HTML элементы table, ol, ul, a, p, div, h1-h6, span и другие, а также для чего именно они используются. Нужно знать разницу между строчными и блочными элементами.
  • версии HTML и какие теги в них присутствуют
  • блочная верстка и ее отличия и преимущества перед табличной
  • HTML5, хотя бы основные элементы

CSS:

  • виды селекторов
  • основные свойства CSS -фон, цвет, размер, отступы, шрифт, границы и другие
  • особенные свойства CSS -float, position и другие. Опыт работы с ними
  • версии CSS и какие свойства к ним относятся
  • какие свойства различно работаю в разных браузерах
  • умение делать блочную структуру страницы
  • CSS3, многие свойства из этой версии уже широко используются верстальщиками

Photoshop:

  1. нарезка макета
  2. определение размеров
  3. получение цвета
  4. вырезание графических элементов

JavaScript или jQuery:

В наше время JavaScript очень нужен для верстки сайта. Далее список по уровню важности или желательности.
  1. очень нужно хотя бы знать что это и для чего применяется
  2. скрыть/показать блок (лайтбокс)
  3. анимация - слайдер, скроллер
  4. умение изменить стиль/класс элемента
  5. умение получить/изменить значение элемента

Общие знания верстки:

  1. кроссбраузерность
  2. валидность
  3. семантика

Нужно понимать эти понятия и знать как их добиться при верстке сайта.

Кроме всего этого, вы должны уметь пользоваться элементами разработчика - просматривать HTML и CSS код, пользоваться консолью.

Очень желательно уметь верстать шаблоны для популярных CMS -- Joomla!, WordPress. Там очень много своих особенностей, ведь шаблон должен быть динамичный.

Не забывайте читать форумы и блоги. В сети есть очень много полезных ресурсов по верстке. Например: Полезные ресурсы для верстальщика.
